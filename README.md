# 💰 BudgetMaster

Умный планировщик ваших финансов на React + TypeScript

## 🚀 Возможности

- 📊 Отслеживание доходов и расходов
- 🤖 AI-аналитика финансов
- 📈 Визуализация расходов по категориям
- 💾 Локальное сохранение данных
- 📱 Адаптивный дизайн

## 🛠 Технологии

- React 18
- TypeScript
- Vite
- CSS3 с переменными
- Local Storage API
